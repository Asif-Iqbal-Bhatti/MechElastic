Tutorials
=========


The following are several examples to explain the usage of MechElastic. Example files required to run these calculations are available in the ``/examples`` directory in the github repo. 

Remember to set the flag ``code`` for the relevant DFT code.

MechElastic has two modes: stand-alone and library. The library mode is more updated and contains all the features in MechElastic.
